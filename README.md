\# 📊 Amazon Sales Data Analysis 



\## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on Amazon sales data to uncover meaningful insights related to sales performance, customer behavior, product trends, and operational efficiency. The analysis was conducted using Python to clean, preprocess, and visualize the data, transforming raw data into actionable business insights.



\---



\## 🎯 Objectives

\- Analyze overall sales trends over time  

\- Identify top-performing product categories  

\- Understand customer purchasing behavior  

\- Explore geographical distribution of sales  

\- Evaluate operational performance through order status and cancellations  

\- Generate actionable business insights  



\---



\## 🛠️ Tools \& Technologies

\- \*\*Python\*\*  

\- \*\*Pandas\*\* (Data Cleaning \& Analysis)  

\- \*\*NumPy\*\*  

\- \*\*Matplotlib \& Seaborn\*\* (Data Visualization)  

\- \*\*Jupyter Notebook\*\*



\---



\## 📂 Dataset Description

The dataset contains Amazon sales transaction data, including:

\- Order details (Order ID, Date, Status)  

\- Product information (Category, Size, Quantity)  

\- Sales metrics (Amount, Currency)  

\- Fulfillment details  

\- Customer location (City, State, Postal Code)  



\---



\## 🔧 Data Cleaning \& Preprocessing

\- Handled missing values in key columns  

\- Standardized date format for time-based analysis  

\- Removed invalid entries (e.g., zero transaction amounts)  

\- Created new features such as month and day for analysis  

\- Segmented dataset into:

&#x20; - \*\*Sales Dataset\*\* (for revenue-focused analysis)  

&#x20; - \*\*Full Dataset\*\* (for operational insights)  



\---



\## 📊 Exploratory Data Analysis (EDA)

The analysis includes:

\- Monthly sales trend analysis  

\- Product category performance  

\- State and city-wise sales distribution  

\- Customer order patterns by day of the week  

\- Order status and cancellation analysis  



\---



\## 📈 Key Insights

\- Sales peaked in \*\*April\*\*, followed by a decline  

\- \*\*T-shirts\*\* are the top-performing category  

\- \*\*Maharashtra, Karnataka, and Telangana\*\* lead in sales  

\- Metro cities like \*\*Bengaluru, Hyderabad, and Mumbai\*\* dominate  

\- \*\*Sunday\*\* has the highest orders, while \*\*Thursday\*\* has the lowest  

\- High order completion rate (>85%) indicates strong operations  



\---



\## 💡 Business Recommendations

\- Focus on high-performing categories to maximize revenue  

\- Improve or reevaluate low-performing product segments  

\- Target high-performing regions with marketing efforts  

\- Boost sales during low-demand periods (midweek)  

\- Maintain efficient fulfillment processes  



\---



\## 🧠 Conclusion

The analysis reveals strong sales performance driven by key categories and regions, along with efficient operational processes. The insights can help businesses optimize product strategy, improve customer engagement, and enhance decision-making.
